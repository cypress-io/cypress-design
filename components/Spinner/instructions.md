# Spinner — Props, Events & Slots

## Install

```bash
yarn add @cypress-design/vue-spinner        # Vue
yarn add @cypress-design/react-spinner      # React
```

## Props

| Prop    | Type                | Default   | Description                                     |
| ------- | ------------------- | --------- | ----------------------------------------------- |
| `theme` | `"light" \| "dark"` | `"light"` | Color theme — use `"dark"` on light backgrounds |
| `size`  | `string \| number`  | `"24"`    | Diameter in px                                  |
