# Logo — Props, Events & Slots

## Install

```bash
yarn add @cypress-design/vue-logo        # Vue
yarn add @cypress-design/react-logo      # React
```

## Props

| Prop      | Type                                | Default     | Description                       |
| --------- | ----------------------------------- | ----------- | --------------------------------- |
| `variant` | `"default" \| "mark" \| "wordmark"` | `"default"` | Which logo asset to render        |
| `theme`   | `"dark" \| "light"`                 | `"dark"`    | Color theme (dark bg vs light bg) |
| `size`    | `"sm" \| "md" \| "lg"`              | `"md"`      | Logo size                         |
