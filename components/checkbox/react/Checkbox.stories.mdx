## Usage

Install the component

```
yarn add @cypress-design/react-checkbox
```

In the script, import the component

```js
import Checkbox from '@cypress-design/react-checkbox'
```

then in your render function, use it

```jsx
export const MyComp = () => {
  return (<div>
    <Checkbox checked={checked} label="Checked" color="indigo" onChange={setChecked}/>
  </div>)
}
```
