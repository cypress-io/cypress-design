## Usage

Install the component

```
yarn add @cypress-design/vue-checkbox
```

In the script, import the component

```js
import Checkbox from '@cypress-design/vue-checkbox'
```

then in the template, use it

```html
<Checkbox checked label="Checked" color="indigo"/>
```

or with the v-model

```html
<Checkbox v-model="checked" label="Checked" color="indigo"/>
```
