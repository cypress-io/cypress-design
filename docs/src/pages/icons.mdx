# Icons

All icons available to Cypress developers. Most icons accept `strokeColor` and `fillColor` props.

Legend for customizable color channels:

- `s` — `strokeColor` available
- `f` — `fillColor` available
- `f+` — `secondaryFillColor` available
- `s+` — `secondaryStrokeColor` available

For full usage details see the [Icon component page](/components/Icon).
