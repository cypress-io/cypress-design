# Card

## Usage

There is no card component — only a CSS class.

```html
<div class="card h-20 w-32 flex items-center justify-center">hello</div>
```

Use `card` on any element to apply the standard bordered surface styles. Cards can also add more shadow for a marketing context:

```html
<div class="card shadow-marketing-card h-20 w-32 flex items-center justify-center">
  hello
</div>
```

`shadow-marketing-card` adds a heavier drop shadow appropriate for marketing pages.
