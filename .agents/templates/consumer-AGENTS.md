# Agent guidance

This project consumes the **Cypress Design System** (`design.cypress.io`).

## Before any UI / visual / content work

Fetch the design system router and follow it to the specific guidance files needed for the task at hand:

```
https://design.cypress.io/agents/index.md
```

The router lists pillars (colors, typography, spacing, iconography, voice, content, personas) and components, each with a one-line trigger. **Fetch only what the task needs** — don't preload everything.

## Before claiming a task is done

Run the self-review checklist against your own diff:

```
https://design.cypress.io/agents/review-checklist.md
```

Fix violations before committing. The checklist covers tokens, spacing, voice, accessibility, and component reuse.
